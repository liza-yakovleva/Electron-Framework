#  Electron Survey System 

**Дисципліна:** Розробка інтерфейсів користувача
**Виконала:** Яковлєва Єлизавета Дмитрівна, студентка 4-го курсу  
**Тема:** Лабораторна робота №2. Розробка десктопного застосунку на базі Framework Electron

---

##  Опис проєкту
Проєкт являє собою сучасний десктопний застосунок для проведення комплексного опитування користувачів. Застосунок побудований на базі **Electron Framework**, що дозволяє поєднувати гнучкість веб-технологій (HTML5, CSS3, JavaScript) із потужністю системного середовища Node.js.

###  Ключові особливості:
- **Комплексна анкета:** Реалізовано 6 типів питань (текстові поля, вибір віку, списки, радіокнопки 1-5, множинний вибір модулів).
- **Якісний UX/UI:** Інтерфейс розроблений за принципами сучасного дизайну з використанням Flexbox для точного позиціонування елементів.
- **Node.js Integration:** Пряма взаємодія з файловою системою для надійного збереження даних.
- **Темна тема:** Стилізація у форматі "Modern Dark", що зменшує навантаження на зір та відповідає трендам професійного ПЗ.

---

## 🛠 Технологічний стек
- **Framework:** Electron
- **Runtime:** Node.js (v24.13.1)
- **Frontend:** HTML5, CSS3 (Flexbox Layout)
- **Backend:** JavaScript (File System API)

---

##  Структура проєкту
- `main.js` — головний процес Electron, керування життєвим циклом вікна.
- `renderer.js` — логіка збору даних, валідація та взаємодія з модулем `fs`.
- `index.html` — декларативна розмітка інтерфейсу.
- `style.css` — стилізація компонентів та сітка Flexbox.
- `survey_electron.txt` — файл, куди експортуються результати опитування.
